子串划分字典树节点数和

DNEGEL3125 Lv2
  2025-05-12 10:31:28 2025-05-12 10:31:28 Created   2025-05-17 09:59:21 2025-05-17 09:59:21 Updated  

题目链接：https://codeforces.com/group/OcmZ7weh45/contest/469866/problem/D

time limit per test: 2 seconds
memory limit per test: 1024 megabytes

给定一长度为 n 的字符串 S，现需将其划分为若干个非空子串，并将划分所得的所有子串插入一棵字典树中。

求：对于所有划分方案，得到的字典树节点个数之和。答案可能很大，只需求出其对 取模的结果。

Input

第一行一个正整数 ()，表示输入字符串的长度。

第二行一个仅包含小写英文字母的，长度为 的字符串 S。保证 S 仅包含小写英文字母。

Output

一个整数，也就是答案。

Examples

input output
2
ab		 6
input output
4
abba		 34
